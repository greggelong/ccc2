Coding Basics

Setup Function

Definition: Called once when the program starts to define initial environment properties and load media.

Only one setup() function per program.


function setup() {
    createCanvas(400, 400);
    background(200);
}

Chinese Translation:


setup() 函数将在程式开始时被调用一次。它可在程序开始时被用来定义初始的环境属性如荧幕大小、背景颜色及媒体加载如图像及字体。每个程序只能有一个 setup() 函数并且他不能在一开始执行后再次被调用。

Draw Function

Definition: Called directly after setup(), continuously executes the code within its block until the program is stopped.


function draw() {
    ellipse(50, 50, 80, 80);
}

Chinese Translation:


在 setup() 之后被调用，draw() 函数将持续地重复执行其中的代码直到该程式终止

Vocabulary Time

Instructions for Drawing Shapes and Lines

Materials Needed: Paper, colored pencils or markers, and a ruler.

Drawing Instructions:

  • Shape 1: Draw a shape with specific dimensions at the given position. Determine the fill color.
  • Shape 2: Draw a shape with specific dimensions at the given position. Determine the fill color.
  • Shape 3: Draw a shape with specific dimensions at the given position. Determine the fill color.
  • Shape 4: Draw a shape with specific dimensions at the given position. Determine the fill color.
  • Shape 5: Draw a shape with specific dimensions at the given position. Determine the fill color.
  • Line 1: Draw a line between the specified points. Determine the stroke color.
  • Line 2: Draw a line between the specified points. Determine the stroke color.

Check Your Work: Students can copy the provided code into a P5.js editor (such as the P5.js Web Editor). Run the code to see if their hand-drawn shapes and colors match the ones generated by the code.

P5.js Code


function setup() {
  // Set up the canvas size
  createCanvas(400, 400);
  
  // Set the background color to white
  background(255);

  // Draw shapes and lines
  // Shape 1
  fill(255, 0, 0); // Fill color (identify the color yourself)
  ellipse(100, 100, 50, 50); // Draw a shape (figure out the type and size)

  // Shape 2
  fill(0, 0, 255); // Fill color
  rect(200, 50, 100, 150); // Draw a shape

  // Shape 3
  fill(0, 255, 0); // Fill color
  ellipse(50, 300, 80, 40); // Draw a shape

  // Shape 4
  fill(255, 255, 0); // Fill color
  ellipse(100, 200, 60, 30); // Draw a shape

  // Shape 5
  fill(128, 0, 128); // Fill color
  rect(250, 250, 80, 80); // Draw a shape

  // Line 1
  stroke(0); // Stroke color
  line(50, 350, 350, 350); // Draw a line

  // Line 2
  stroke(0); // Stroke color
  line(200, 50, 200, 350); // Draw a line
}

function draw() {
  // Stop the draw loop since we don't need to redraw the shapes
  noLoop();
}
